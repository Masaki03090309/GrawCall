## タスク管理方法

### 優先度

- **P0**: クリティカル（システムの根幹）
- **P1**: 高優先度（主要機能）
- **P2**: 中優先度（補助機能）
- **P3**: 低優先度（Nice to have）

### 見積もり

- **S**: 1-2時間
- **M**: 半日-1日
- **L**: 2-3日
- **XL**: 4-5日
- **XXL**: 1週間以上

### タスク形式

```
[優先度] [見積もり] タスク名
  - 詳細手順1
  - 詳細手順2
  担当: <担当者>
  依存: <依存タスク>
```
